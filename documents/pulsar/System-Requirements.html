1 System Requirements

  • Any operating systems that can run the other subsystems that are described here.
  • Java 8
  • JNA Java Native Access
  • Jack Audio Connection Kit

Pulsar has been developed and tested in Ubuntu. A cursory experiment to run Pulsar in Windows 10 with Windows JACK was succeeded. It is still unknown that Pulsar can run in OS X and further experiments are needed.
